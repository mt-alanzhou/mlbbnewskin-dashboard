# MLBB New Skin Feedback Dashboard

This folder is ready to publish with GitHub Pages.

## Files

- `index.html`: GitHub Pages entry point.
- `mlbbnewskin-dashboard.html`: auto-refreshing dashboard updated by the Codex automation.

## GitHub Pages Setup

1. Create a public GitHub repository.
2. Upload these files to the repository root.
3. In GitHub, open `Settings` > `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select the main branch and root folder.
6. Save, then open the GitHub Pages URL GitHub provides.

The dashboard page refreshes every 10 minutes. The automation updates `mlbbnewskin-dashboard.html`.
